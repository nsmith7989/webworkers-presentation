Grunt

What?

Grunt is built on Node.js

nodejs.org

Javascript Environment that can interact with your local file system

Anything node can do, Grunt can do.

Grunt is a system for running node scripts.

Grunt

Installation.

Install Steps

  1. Install Node nodejs.org
  2. Install Grunt npm install -g grunt-cli

Usage Steps

Grunt configuration is per project

Based on two files

  1. package.json
  2. Gruntfile.js

Get these by using a generator e.g. npm install grunt-init -g

or by manually creating them...

package.json

{
  "name": "WebWorkers-Gruntification",
  "version": "0.1.0",
  "devDependencies": {
    "grunt": "~0.4.2",
    "grunt-contrib-uglify": "~0.2.2",
    "grunt-sass": "~0.12.0",
    "browser-sync": "^0.7.3"
  }
}

npm install grunt-contrib-watch --save-dev

Gruntfile.js

module.exports = function(grunt) {

  // Project configuration.
  grunt.initConfig({

  uglify: {
      //tasks options/config here
    }
   
  });
  //load tasks here
  grunt.loadNpmTasks('grunt-contrib-uglify');

  //name tasks here
  grunt.registerTask('default', ['concat','uglify']);

};

Responsive Wordpress Theme (Zurb Foundation)

What We Want (development)

  • Sass Compilation (grunt-sass)
  • Automatic Browser Prefixing (autoprefixer)
  • Font Face Tracking (fontface)
  • Live Reloading (browserSync)

What We Want (production)

  • Smallest Files Possible
    • Minified CSS (grunt-sass)
    • Minified and Concatinated JS (concat/uglify)
    • Smallest Site asset images as possible (imagemin)
